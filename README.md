#  ميكرو مصر - نظام إدارة النقل الذكي

##  عن المشروع
نظام متكامل لإدارة شركات النقل البري، يشمل إدارة السائقين، الرحلات، المحطات، المدفوعات، والتقارير.

##  المميزات
-  إدارة السائقين مع نظام مكافآت للشركات المتعاقدة
-  تتبع الرحلات والحالة في الوقت الفعلي
-  خريطة تفاعلية للمحطات
-  لوحة تحكم بإحصائيات ورسوم بيانية
-  نظام مدفوعات متكامل
-  واجهة عربية بالكامل (RTL)

##  التقنيات المستخدمة
- React 18
- React Router v6
- Axios
- Framer Motion
- Chart.js / Recharts
- React Leaflet
- React Hot Toast
